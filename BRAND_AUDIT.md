# 브랜드 감사 결과

## 기준

- 한글: 따란
- 영문: T'ARAN
- 코드 식별자: `taran`
- 저장소 키 접두사: `taran-`

## 적용

- 브랜드 값은 `scripts/core/brand.js`에서 관리합니다.
- 새 공개 핵심 페이지와 관리자 화면은 따란 표기를 사용합니다.
- 이전 저장 키는 `scripts/core/storage.js`가 값이 있는 경우에만 새 키로 한 번 복사합니다.
- 새 저장은 `taran-` 키로만 이루어집니다.

## 의도적으로 남긴 이전 명칭

이전 명칭은 저장 데이터 마이그레이션 키 목록에만 남아 있습니다. 이 문자열은 사용자 화면에 출력되지 않으며 기존 브라우저 데이터를 잃지 않기 위해 필요합니다.

## 남은 범위

아직 새 디자인 시스템으로 전환하지 않은 레거시 보조 페이지의 내부 클래스명과 데이터 파일명은 별도 단계에서 제거합니다. 공개 핵심 페이지에는 타 서비스가 연상되는 `jobplanet`, `jp-` 명칭을 사용하지 않습니다.
