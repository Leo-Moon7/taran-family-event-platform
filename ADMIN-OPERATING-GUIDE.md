# T'ARAN 관리자 운영 안내

T'ARAN(따란)은 프리미엄 가족행사 큐레이션 플랫폼입니다. 관리자 화면은 개발자가 아닌 운영자가 매일 확인하고 수정할 수 있도록 `고객 반응`, `프로모션`, `업체 정보`, `준비백과` 중심으로 구성합니다.

## 1. 운영자가 매일 보는 순서

1. 오늘 할 일
   - 견적 공유 검토
   - 업체 정보 보강 대기
   - 진행 중인 배너
   - 최근 저장 상태
2. 프로모션
   - 메인 배너
   - 업체 목록 배너
   - 준비백과 배너
3. 고객·견적
   - 견적 공유 자료
   - 포인트 지급 대기
   - 고객 상태
4. 업체 관리
   - 신규 업체 등록
   - 업체 정보 보강
   - 공개 전 기준 체크
5. 준비백과
   - 글 작성
   - 글 보강
   - SEO용 질문 정리
6. 통계
   - 방문
   - 업체 클릭
   - 견적 공유
   - 준비백과 조회

## 2. 업체 공개 기준

업체는 아래 기준 중 3개 이상 충족하면 공개 후보로 봅니다.

- 주소 또는 지역이 확인됨
- 가능한 행사 유형이 확인됨
- 참고 후기 또는 공식 채널이 있음
- 가격 또는 수용인원 중 최소 1개 이상 확인됨

기준이 부족하면 `검토 대기` 상태로 두고, 고객에게는 아직 노출하지 않는 방향을 권장합니다.

## 3. 고객 제보와 개인정보 운영 원칙

고객이 공유한 견적서와 업체 정보는 업체 조건표를 보강하는 데 사용합니다.

- 이름, 전화번호, 주소, 계좌번호 등 개인정보는 가린 자료만 받습니다.
- 원본 파일은 검토 후 보관하지 않는 정책을 기본으로 합니다.
- 고객DB는 공개 사이트에 포함하지 않습니다.
- 공개 파일 다운로드에는 업체·글·배너·문구만 포함합니다.

## 4. 온라인 저장 연결

초기에는 로컬 미리보기와 파일 다운로드 방식으로 운영할 수 있습니다. 여러 명이 함께 관리하거나 실제 로그인 기반 운영이 필요해지면 Supabase 저장 방식으로 전환합니다.

관리자 화면의 `고급 설정`에서 아래 값을 붙여넣을 수 있습니다.

- Supabase Project URL
- Supabase anon public key

`service_role key`는 절대 브라우저나 공개 파일에 넣지 않습니다.

## 5. 개발자용 기능

아래 기능은 일반 운영 화면에서 숨겨두고, 필요할 때만 고급 설정에서 사용합니다.

- 공개 반영 파일 다운로드
- 관리자 백업 파일 다운로드
- 온라인 저장 내용 불러오기
- DB 설치 파일 다운로드
- AI/외부 데이터 가져오기
- 현재 저장될 데이터 미리보기
