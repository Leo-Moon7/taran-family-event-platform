# MKT-001 전문 작업 요청문

작업 ID: MKT-001

작업 목적은 현재 공개 경로와 기존 콘텐츠를 기준으로 SEO·콘텐츠 공백의 내부 기준선을 만드는 것입니다. 사이트와 외부 채널은 수정하지 마십시오.

## 먼저 읽을 문서

1. `AGENTS.md`
2. `ops/TASK_SPECS.md`의 MKT-001 카드
3. `docs/01_사업정의.md`
4. `docs/02_현재사이트분석.md`
5. `docs/08_마케팅전략.md`
6. `docs/09_운영정책.md`
7. `sitemap.xml`, `robots.txt`

## 수정 범위

- 수정 허용 경로: `ops/reports/MKT-001-seo-content-gap.md` 신규 1개
- 수정 금지 경로: 위 파일 외 전부. 특히 공개 HTML, `sitemap.xml`, `robots.txt`, 기존 전략 문서, 콘텐츠 데이터, 제품 코드, 외부 채널

## 구현 범위

- 공개 HTML의 title, description, canonical, OG, JSON-LD, sitemap 포함 여부를 읽기 전용으로 조사합니다.
- 기존 준비백과·블로그 주제와 중복을 확인합니다.
- 지역×행사×검색 의도 공백을 정리하되 전국·8개 행사 범위와 서울 돌잔치 초기 우선순위를 변경하지 않습니다.
- 각 결과를 `현재 사실`, `개선 후보`, `사용자 승인 또는 데이터 선행`으로 분리합니다.

새 페이지·글·메타태그를 작성하거나 외부 게시·광고·검색엔진 제출을 하지 마십시오. 범위 밖 문제는 수정하지 말고 `신규 작업 후보`로만 기록하십시오.

## 완료 조건과 테스트

- 공개 경로 인벤토리, 기술 SEO 공백, 콘텐츠 중복, 초기 우선 후보, 선행 조건이 표로 정리되어야 합니다.
- 공개 HTML 수와 sitemap 항목을 교차 대조합니다.
- 기존 콘텐츠 제목 중복과 8개 행사 범위를 확인합니다.
- `rg` 등 읽기 전용 검사만 사용하며 빌드·패키지·제품 테스트를 변경하지 않습니다.
