# FE-011 전달문 — 승인된 최종 홈·준비 도구·비교 화면 구현

## 작업 목적

사용자가 2026-07-23 전체 승인한 최종 디자인을 기존 손품해방 구조에 구현한다. 홈은 업체 검색 → 비용 계산 → 검증 가능한 업체 탐색 → 비교·견적 흐름을 우선하고, 체크리스트는 별도의 행사 일정 관리 도구로 유지한다.

## 구현 범위

- 승인된 홈 8개 섹션 순서와 공통 메뉴
- 검색 기본값 `서울특별시 + 돌잔치·백일`과 기존 검색 쿼리
- 계산기 행사별 항목, 범위 결과, 업체 필터 전달
- 체크리스트 행사 설정, 일정 관리, 기능별 연결
- 비교함 최대 3개, 실제 선택 수 CTA, 교체·삭제, 모바일 가로 스크롤
- 공개 업체가 없을 때 가짜 카드 없이 검증 기준과 빈 상태
- 390/768/1440px 반응형·키보드 접근성

## 범위 밖 문제 처리

범위 밖 문제를 직접 수정하지 말고 파일·재현 경로·영향과 필요한 변경 요청을 총괄 PM에게 보고한다. 공통 파일, API, DB, 라우팅, 환경변수, 패키지 변경이 필요하면 즉시 중단한다.

## 완료 조건

승인된 흐름과 문구가 반영되고 모든 CTA가 실제 기존 경로로 이동한다. 가짜 업체·평점·후기·가격과 죽은 버튼이 없어야 한다. 계산 조건 전달, 체크리스트 일정 관리, 비교 선택 수 CTA가 재현되어야 하며 3개 뷰포트에 가로 넘침과 콘솔 오류가 없어야 한다.
