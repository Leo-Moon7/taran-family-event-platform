# FE-009 전달문 — 베이지 C안 홈 안전 축소 구현

현재 상태는 `DONE`이다. D-22 승인과 OPS-009 격리·PM 검수 `PASS` 후 순차 실행했고, 결과는 `ops/reports/FE-009-safe-beige-home.md`에 기록했다. 최종 배포는 금지 상태다.

## 작업 목적

승인된 베이지 C안의 시각 방향을 실제 홈에 구현하되, 검증되지 않은 업체·후기·가격과 준비되지 않은 개인정보·업체 참여 기능을 노출하지 않는다. 방문자가 첫 화면에서 행사 검색, 계산기, 체크리스트, 준비백과와 정보 나눔을 바로 사용할 수 있게 한다.

- 첫 화면의 행사·지역·인원 검색과 기존 쿼리 계약
- 기존 8개 행사 유형 빠른 시작

## 구현하지 않을 범위

- 업체 카드·업체 수·지역별 수, 후기·평점·추천·가격
- 신뢰 라벨의 실제 업체 부착
- 비교·견적 문의 강조
- 고객 정보 수정 제안, 업체 권한 요청, 신규 업체 등록
- 생성 시안 이미지를 운영 자산으로 직접 복사

범위 밖 문제는 직접 수정하지 말고 재현 경로와 영향을 총괄 PM에게 보고한다.
