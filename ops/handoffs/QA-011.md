# QA-011 전달문

## 작업 목적

서울 돌잔치 업체 후보 발견과 영업 상태 확인에 사용할 수 있는 공공데이터 원천을 공식 자료로 조사하고, 데이터셋별 이용허락·출처·갱신·허용 필드·중단 조건을 레지스트리로 작성한다.

## 수정 금지 경로

- 위 보고서 외 모든 파일
- API 활용신청·키 사용·데이터 호출/다운로드/저장
- 제품 코드, DB, `backend/data`, 환경변수, 패키지·잠금 파일, CHG-A~C

## 구현 범위

- 후보 데이터셋별 제공기관, dataset ID, 공식 URL, 확인일
- 공간·업종 범위, 갱신주기, 이용허락, 상업 이용, 변경, 출처표시, 제3자 권리
- 사용할 수 있는 필드와 행사 가능·가격·사진·문의 의사로 확장 해석하면 안 되는 필드
- LOCALDATA 종료/전환 여부와 실제 현재 endpoint
- 불확실·종료·권리 미확인 원천의 사용 중단 조건

범위 밖 문제는 직접 수정하거나 활용신청하지 말고 `신규 후보` 또는 `사용자 결정 필요`로 보고한다. 공식 제공기관·공공데이터포털 등 1차 자료를 우선하고 승인된 범위를 확대하지 않는다.

## 완료·테스트

- 완료 조건: 모든 제안 원천에 공식 URL·확인일·권리·필드·갱신·출처·중단 조건이 있으며 미확정 원천을 사용 가능으로 표시하지 않음
- 테스트: 링크 접근·dataset ID·중복·종료 상태·필수 열 검사, 공공데이터포털 이용정책과 각 데이터셋 이용허락 교차 확인
