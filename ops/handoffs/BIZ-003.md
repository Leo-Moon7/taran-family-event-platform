# BIZ-003 전달문

## 작업 목적

## 구현 범위

- 돌잔치 장소/서비스의 포함·제외·확인 질문표
- 가격·인원·주차·시설의 단위·조건·결측·유효기간 사전
- `공공데이터 등록`, `정보 확인 중`, `업체 직접 제출`, `사업자 확인`, `관리자 검수`, `최근 정보 확인 필요`의 적용 조건과 금지 문구
- 기본 페이지·비교·견적 문의 진입 최소 자격
- 경계 사례와 서울 파일럿 판단 지표

범위 밖 문제는 직접 고치지 말고 기존 백로그·위험과 중복 여부를 확인해 `신규 후보`로만 보고한다. 승인된 범위를 임의로 확대하지 않는다.

## 완료·테스트

- 완료 조건: 분류 질문표, 비교 사전, 신뢰 축, 화면 문구, 최소 자격, 10개 이상 경계 사례, 승인 필요 항목이 재현 가능하게 기록됨
- 테스트: ADR-016·BIZ-002·BE-005와 모순 검사, 결측·만료·사업자 확인·업체 제출·관리자 반려 시나리오 검사, 미확정 거래·가격·추천 과장 표현 0건
