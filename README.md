# 노품 디자인 업그레이드 시제품

기존 기능을 유지하면서 섹션별 디자인 시안을 참고해 시각 완성도와 반응형 사용성을 높인 정적 웹 시제품입니다.

## 포함 기능

- 서울 전역 돌잔치 장소 조건 검색
- 보증 인원·주차·단독룸 우선 필터와 페이지 최종 갱신일
- 돌잔치 총예산 계산기
- 예산 구간별 준비 카테고리 추천
- 저장되는 상세 준비 체크리스트
- 계산기와 체크리스트 독립 페이지
- 행사일 기준 현재 준비 구간 안내
- 이메일 로그인과 관심 업체·예산·체크리스트 계정 저장
- 과업별 관련 업체·준비물 연결 영역
- 업체 전체보기 별도 페이지
- 업체별 숨은 비용·운영 조건 상세정보와 관심 업체 저장
- 돌잔치 준비 에디토리얼 가이드 페이지
- 파트너 어드민·에스크로·검증 후기 장기 기획 페이지
- 크롤링 승인 대기 → 관리자 검수 → 발행 상태 시제품
- 사업자 확인 기반 업체 소유권 요청(Claim Listing)
- 승인된 업체만 열리는 정보 수정 대시보드
- Basic 무료·Premium 광고 상품 선택 화면
- 이름·이메일·전화번호 기반 시제품 회원가입 화면
- 이름·이메일·전화번호 확인 및 필수/선택 동의 분리
- 로그인 후 관심 업체를 모아보는 내 정보 화면
- 독립 이미지 에셋을 활용한 장소·준비 콘텐츠 카드
- 섹션별 스크롤 등장 효과와 모바일 최적화
- 모바일 반응형 화면

## 폴더

- `assets/images`: 사이트에서 실제 사용하는 독립 이미지 에셋
- `design-references`: 코드 구현 기준으로 사용한 8개 섹션별 가로 시안

현재 표시되는 업체명과 가격은 모두 기능 확인용 예시이며 실제 데이터가 아닙니다. 회원 정보는 현재 브라우저에만 저장되며 실제 이메일 인증과 서버 저장은 아직 연결되지 않았습니다.

`admin-review.html`은 권한 흐름을 확인하기 위한 로컬 시제품입니다. 실제 사이트에 공개하기 전 관리자 인증, 역할 기반 접근 제어, 사업자 문서 암호화 저장·파기, 수정 이력 로그를 반드시 구현해야 합니다.

`backend` 폴더에는 SQLite DB, 공개 업체 API, 관리자 검수 API, 업체별 수정 권한 API의 로컬 MVP 기반이 포함되어 있습니다. 실행과 보안 주의사항은 `backend/README.md`와 `backend/SECURE-DEPLOYMENT-CHECKLIST.md`를 확인합니다.

`index.html`을 브라우저로 열면 별도 설치 없이 작동합니다.
